# Asteroid Danger Detector

Detecting potentially dangerous asteroids using NASA data.

## Dataset
Download the asteroid dataset from:
https://www.kaggle.com/datasets/

Place the CSV file in the project folder and rename it to `d.csv`

## How to Run
1. Install requirements: pip install streamlit pandas pymysql cryptography
2. Run load_db.py to load data into MySQL
3. Run streamlit run app.py

## Tech Stack
- Python
- Streamlit
- Pandas
- MySQL
